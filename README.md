# Newsletter sign-up form with success message

This is a solution to the "Newsletter sign-up form with success message" challenge from Frontend Mentor.  
The project is built with React, Vite and Tailwind CSS.

## Features

- Users can enter their email and submit the form
- Inline validation:
  - Shows an error if the field is empty
  - Shows an error if the email address is not correctly formatted
- Error state styling:
  - Red border and light red background for invalid input
  - Error message displayed next to the label
- Success screen:
  - Shows a confirmation message with the submitted email
  - "Dismiss message" button resets the form
- Smooth transition between form and success state
- Responsive layout:
  - Mobile and desktop designs
  - Different illustration images for mobile and desktop
- Hover and focus states for interactive elements

## Tech stack

- React (with Vite)
- Tailwind CSS
